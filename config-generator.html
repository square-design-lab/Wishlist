1 Create Wishlist Page

          
In Squarespace, go to Pages → + → Blank Page. Set the URL slug to /wishlist (or your chosen slug). Add a Code Block to the page — the plugin will inject the wishlist content inside it.

        

        

          
2 Add a Form Block

          
Add a Form Block to your site footer (so it's available on all pages). Add a textarea field and set its label to match the Form Field Name in your config (default: Wishlist). Set the form to open in a Lightbox. The plugin will auto-fill this field when users click "Send Wishlist".

        

        

          
3 Configure the Plugin

          
Use the options on the left to customise the heart icon appearance, wishlist page labels, and form settings. The preview on the right updates in real time.

        

        

          
4 Copy & Paste Code

          
Go to the Code Output tab and copy the generated code. In Squarespace, go to Settings → Advanced → Code Injection → Header and paste the code.

        

        

          
5 How It Works

          
Heart icons appear on product images in the store page and on product detail pages. Clicking a heart adds the product to the wishlist (stored in the visitor's browser). The header heart icon shows a badge count and links to the wishlist page. On the wishlist page, users can view saved items, add them to cart, remove them, or send the entire wishlist to you via the popup form.
